# Quadcopter_program
四轴无人机飞控程序
飞控主程序JAVA 用于Raspberry Pi3 modelB

串口连接Teensy 3.6 用于生成PWM信号

手机遥控程序Android 可远程遥控无人机 实现远程起飞，四轴转速控制
